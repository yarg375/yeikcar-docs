## Account Setup

To start using YeikCar, you'll need to set up your account by following these steps:

1. **Account Registration**
   - Access the registration portal
   - Complete the required information
   - Verify your email address

2. **Basic Settings**
   - Configure your company profile
   - Set up notification preferences
   - Customize your dashboard

3. **Security Configuration**
   - Set a secure password
   - Configure two-factor authentication
   - Review user permissions

## Next Steps

Once you've completed the initial setup, you can continue with the [first steps guide](/first-steps) to start using YeikCar's main features.
