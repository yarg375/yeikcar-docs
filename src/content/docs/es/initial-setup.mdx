## Configuración de tu Cuenta

Para comenzar a utilizar YeikCar, necesitarás configurar tu cuenta siguiendo estos pasos:

1. **Registro de Cuenta**
   - Accede al portal de registro
   - Completa la información requerida
   - Verifica tu correo electrónico

2. **Ajustes Básicos**
   - Configura tu perfil de empresa
   - Establece tus preferencias de notificación
   - Personaliza tu dashboard

3. **Configuración de Seguridad**
   - Establece una contraseña segura
   - Configura la autenticación de dos factores
   - Revisa los permisos de usuario

## Próximos Pasos

Una vez completada la configuración inicial, puedes continuar con la [guía de primeros pasos](/es/first-steps) para comenzar a utilizar las funciones principales de YeikCar.
